# Repo Analyzer Skill

## Overview
**Purpose**: Analyze repository structure, tech stack, complexity, tech debt indicators.
**Target Users**: CTO (primary), CEO (technical overview)
**Capabilities**: Language breakdown, dependency analysis, code complexity metrics, tech debt scoring

## Required Tools
`Glob`, `Read`, `Bash` (cloc, git), `Write`

## Usage
```
"Используя repo-analyzer skill, analyze Sales AI codebase"

→ Detects: Python, JavaScript, HTML/CSS
→ Analyzes: package.json, requirements.txt
→ Metrics: Lines of code, file count, dependencies
→ Tech debt: TODO count, code duplication, outdated deps
```
