# Repo Agent Suggester Skill

## Overview
**Purpose**: Recommend AI agents based on repository type and project needs.
**Target Users**: CEO, CTO
**Capabilities**: Agent-to-task mapping, workflow recommendations

## Required Tools
`Read` (repo structure), `Write` (agent recommendations)

## Usage
```
"Используя repo-agent-suggester skill, recommend agents для Sales AI project"

→ Detects: Python + finance scripts → CFO agent useful
→ Detects: HTML landing pages → UI Designer + Site Audit agents
→ Detects: Marketing materials → CMO agent
→ Generates: Agent assignment recommendations
```
