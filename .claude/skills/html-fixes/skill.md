# HTML Fixes Skill

## Overview
**Purpose**: Быстрые HTML/CSS исправления (responsive issues, accessibility, semantic markup).
**Target Users**: CTO, Frontend developers
**Capabilities**: Fix common HTML/CSS patterns, implement audit findings

## Required Tools
`Read`, `Edit`, `mcp__ide__getDiagnostics`, `Bash` (for validation)

## Usage
```
"Используя html-fixes skill, fix accessibility issues в Sales AI/site/code4.html"
→ Adds alt text, aria-labels, semantic tags
```
