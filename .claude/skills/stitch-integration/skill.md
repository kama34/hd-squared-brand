# Stitch Integration Skill

## Overview
**Purpose**: Helpers for Stitch Data integration (ETL pipelines from SaaS tools to data warehouse).
**Target Users**: CTO, Data Engineer, CFO (for financial data pipelines)
**Capabilities**: Stitch connection setup, integration patterns, troubleshooting

## Required Tools
`Read`, `Write`, `WebSearch` (Stitch docs), `Bash` (API testing)

## Usage
```
"Используя stitch-integration skill, setup Stripe → BigQuery pipeline via Stitch"
→ Provides connection configuration
→ Recommends sync frequency
→ Generates SQL queries for analysis
```
