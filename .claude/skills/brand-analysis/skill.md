# Brand Analysis Skill

## Overview
**Purpose**: Analyze brand positioning and competitive differentiation.
**Target Users**: CMO, CEO, Board
**Capabilities**: Brand perception analysis, positioning gap identification, competitive brand comparison

## Required Tools
`WebSearch`, `Read`, `Write`

## Usage
```
"Используя brand-analysis skill, analyze Sales AI brand positioning vs competitors"
→ Researches competitor brand messaging
→ Identifies Sales AI unique positioning
→ Generates brand differentiation report
```
