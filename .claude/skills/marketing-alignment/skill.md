# Marketing Alignment Skill

## Overview
**Purpose**: Align marketing messaging across channels (website, social, emails, ads).
**Target Users**: CMO (primary), Marketing team
**Capabilities**: Message consistency checker, value prop alignment, tone verification

## Required Tools
`Read`, `Write`, `WebSearch` (competitor messaging)

## Usage
```
"Используя marketing-alignment skill, check message consistency для Sales AI"
→ Compares website, LinkedIn, Twitter, email campaigns
→ Identifies inconsistencies in value proposition
→ Recommends unified messaging framework
```
