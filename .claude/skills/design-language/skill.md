# Design Language Skill

## Overview
**Purpose**: Design system consistency (colors, typography, spacing, components).
**Target Users**: UI Designer, CMO, CTO
**Capabilities**: Extract design tokens, document style guide, check consistency

## Required Tools
`Read`, `Grep` (CSS patterns), `Write` (style guide)

## Usage
```
"Используя design-language skill, document Sales AI design system"
→ Extracts colors, fonts, spacing from CSS
→ Generates style guide documentation
```
